c) First, we need to estimate the variance of the random walk process. We now that the differences are distributed iid: Given \( Y_t -Y_{t-1} \sim N( \beta_0,\sigma^2_{\epsilon}) \) we can estimate \( \sigma^2_{ \epsilon} \) by taking the variance of the differences: 9.2985 × 10-5

Looking at the model: \[ Y_{t+1} = Y_t + \mu + \epsilon_{t+1} \] And recursively substituting: \[ Y_{t+i} = Y_t + i * \mu + \epsilon_t +\epsilon_{t+1} + \cdots + \epsilon_{t+i} \] Given that we assume the error to normally distributed with above variance, a natural 95% confidence interval would be \[ std(Y_{t+i}|Y_t) = sd(\sum_{j=0}{i}{\epsilon_{t+j}}) = \sqrt{k}\sigma_\epsilon \]

Resulting in the following confidence intervals:

So \[ var(lgdp_{T+k} - lgdp_{T}) = var(k*\mu + \epsilon_{T+1} +\epsilon_{T+2} + \cdots + \epsilon_{T+k}) \]
