Mapping tweets

every tweet is a story

Using an 8-core ubuntu virtual machine, we analyzed over 12,000,000 tweets in the greater london area.

In this analysis, we focus on the 720,669 tweets with valid geographic information. By plotting tweets over london and combining them with real estate prices, we can visualize what people are thinking about where, and how it relates to the wealth of the neighborhood.

Tweets & House Prices

Using publically available information on real estate transactions, we can then connect our tweets to characteristics of housing prices

Latent Semantic Analysis

Using machine learning algorithms to link tweet content and real estate prices

SETUP

We aggregate the tweet content over neighborhoods and then create a word-neighborhood matrix of terms

Then, we utilize cutting-edge gamma lasso regression techniques designed to deal with extremely sparse data

GAMMA LASSO

By allowing seperate coeffecients for each neighborhood, we can track and summarize the quintessential tweet vocabulary in each locale, as well as use the sparse regression techniques to match tweet content with changes in housing prices
